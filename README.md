# futuristic_hud_weather_news

Futuristic HUD Weather & News with AI Assistant.

## Getting Started

Futuristic HUD Weather & News is a cutting-edge Flutter mobile app featuring real-time weather, 
global news, and AI-powered chat assistance in a stunning HUD-style interface with retro monospace typography. 
Seamlessly integrates OpenWeatherMap, NewsAPI.org, and OpenRouter.ai for intelligent AI conversations.
